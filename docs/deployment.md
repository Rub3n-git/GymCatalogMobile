# Despliegue

## Backend — Render
- Servicio: Render
- URL: https://gymcatalog-api.onrender.com
- Root Directory: server
- Build Command: npm install && npm run build
- Start Command: npm start

## Frontend — Vercel
- Servicio: Vercel
- URL: https://gymcatalog.vercel.app
- Variable de entorno: VITE_API_URL

## Variables de entorno
- `VITE_API_URL` — URL de la API en producción
