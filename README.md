# GymCatalog

Aplicación fullstack para explorar y organizar ejercicios de gimnasio.
Permite buscar ejercicios por músculo y dificultad, ver su detalle
y crear una rutina diaria personalizada.

## URLs
- **Frontend:** https://gym-catalog-eight.vercel.app
- **API:** https://gymcatalog-api.onrender.com/api/v1

## Tablero Trello
[Enlace al tablero](https://trello.com/b/FrB9tMCW/mi-tablero-de-trello)

## Tecnologías
- Frontend: React + TypeScript + Tailwind + Vite
- Backend: Node.js + Express + TypeScript
- Despliegue: Vercel + Render

## Arrancar en local

### Backend
```cmd
cd server
npm install
npm run dev
```

### Frontend
```cmd
npm install
npm run dev
```

## Estructura del proyecto
- `src/` — código del frontend
- `server/` — código del backend
- `docs/` — documentación del proyecto
